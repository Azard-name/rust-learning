💡 En Rust, toutes les variables sont immutables par défaut. mut = mutable — déclare explicitement que la variable peut changer. 
String::new
// Fonction qui crée un nouveau String vide.
// Le :: lie String et new (méthode associée au type String)
// → Crée une variable mutable reliée à une valeur String vide
let mut guess = String::new();

    .expect("Failed to read line");
    // Lit la valeur retournée (Result) — si erreur, affiche le message et crash

étape 2

Generating a Secret Number

use std::io;
use rand::Rng; // Crate externe — outil de génération aléatoire

fn main() {
    println!("Guess the number!");

    let secret_number = rand::thread_rng().gen_range(1..=100);
    // Génère un nombre aléatoire entre 1 et 100 inclus (..= = inclusif)

    println!("The secret number is: {secret_number}");
    println!("Please input your guess.");

    let mut guess = String::new();

    io::stdin()
        .read_line(&mut guess)
        .expect("Failed to read line");

    println!("You guessed: {guess}");
}
⚠️ La version du Book ("^0.999.0") n'existe pas. Utiliser rand = "0.8.5" dans Cargo.toml.
étape 3

Comparing the Guess to the Secret Number

use std::cmp::Ordering; // Outil de comparaison (Less, Greater, Equal)
use std::io;
use rand::Rng;

fn main() {
    println!("Guess the number ~ ♪");

    let secret_number = rand::thread_rng().gen_range(1..=100);

    println!("Please input your guess :");
    let mut guess = String::new();

    io::stdin()
        .read_line(&mut guess)
        .expect("Failed to read line");

    let guess: u32 = guess.trim().parse().expect("Please type a number!");
    // u32 = unsigned 32-bit integer (0 à ~4 milliards, pas de négatifs)
    // .trim() supprime les espaces et le \n de fin de ligne
    // .parse() convertit le String en nombre

    println!("You guessed: {guess}");

    match guess.cmp(&secret_number) { // Comparaison des deux valeurs
        Ordering::Less    => println!("Too small!"),
        Ordering::Greater => println!("Too big!"),
        Ordering::Equal   => println!("You win!"),
    }
}

étape 4

Allowing Multiple Guesses with Looping

use std::cmp::Ordering;
use std::io;
use rand::Rng;

fn main() {
    println!("Guess the number ~ ♪");

    let secret_number = rand::thread_rng().gen_range(1..=100);

    loop { // Boucle infinie — tourne jusqu'au break
        println!("Please input your guess :");
        let mut guess = String::new();

        io::stdin()
            .read_line(&mut guess)
            .expect("Failed to read line");

        let guess: u32 = guess.trim().parse().expect("Please type a number!");

        println!("You guessed: {guess}");

        match guess.cmp(&secret_number) {
            Ordering::Less    => println!("Too small!"),
            Ordering::Greater => println!("Too big!"),
            Ordering::Equal   => {
                println!("You win!");
                break; // Brise le loop quand le guess est correct
            }
        }
    }
}
⚠️ Le loop doit contenir read_line ET parse — sinon la boucle tourne avec la même valeur à l'infini.
étape 5

Handling Invalid Input

// Version robuste — remplace .expect() par un match sur Result
let guess: u32 = match guess.trim().parse() {
    Ok(num) => num,       // Si parse réussit → utilise le nombre
    Err(_)  => continue,  // Si erreur de saisie → relance le loop sans crash
};

use std::cmp::Ordering;
use std::io;
use rand::Rng;

fn main() {
    println!("Guess the number ~ ♪");

    let secret_number = rand::thread_rng().gen_range(1..=100);

    loop {
        println!("Please input your guess.");

        let mut guess = String::new();

        io::stdin()
            .read_line(&mut guess)
            .expect("Failed to read line");

        let guess: u32 = match guess.trim().parse() {
            Ok(num) => num,
            Err(_)  => continue,
        };

        println!("You guessed: {guess}");

        match guess.cmp(&secret_number) {
            Ordering::Less    => println!("Too small!"),
            Ordering::Greater => println!("Too big!"),
            Ordering::Equal   => {
                println!("You win!");
                break;
            }
        }
    }
}
✅ Code robuste — gère les inputs invalides, boucle jusqu'à la bonne réponse, s'arrête proprement avec break.
